# A-Data-driven-Approach-to-the-Longitudinal-Study-of-Canine-Vocal-Pattern-Development

## Canine Age Transition Vocalization Dataset

### Dataset Summary

The **Canine Age Transition Vocalization Dataset** is a large-scale, meticulously curated collection of dog vocalizations designed for longitudinal studies. This dataset fills a critical gap in the field by providing precisely verified metadata, including exact birthdates, breed, and individual dog IDs for 125 dogs across 6 common breeds. It is uniquely suited for researchers investigating how canine vocal patterns develop and evolve over a dog's lifetime.

This dataset contains **79,142** bark units (from **55,718** Barkseqs) totaling **11.4 hours** of vocalizations. The in-depth metadata and longitudinal tracking are crucial for analyzing developmental changes in vocal parameters, including defined bark types and Elemental Dog Bark Units (EDBUs), across five distinct life stages.
